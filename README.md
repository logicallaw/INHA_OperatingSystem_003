## Build and Run Instructions

1. Create a build directory:
   ```bash
   mkdir build && cd build
   ```

2. Compile your source code:
   ```bash
   cmake ..
   ```

3. Run make:
   ```bash
   make
   ```

4. Run the executable (e.g., Dining_Philosophers):
    ```bash
    ./Dining_Philosophers
    ```

5. Etc./ shortcut command:
   ```
   cmake .. && make && ./Dining_Philosophers
   ```
